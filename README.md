# Iris （鸢尾）

#### 这个两性社交的生态该叫什么，想了很久，但思来想去用鸢尾来命名挺好。鸢尾花最常用来象征爱情与友谊，也代表了好消息和爱的使者，蕴含着思念、优雅的心、使命，文中的意思也在不同颜色的鸢尾花语中有所体现。感谢能在自然的杰作里找到花语和表达意思如此接近的花，这是件幸运的事。此外，《鸢尾花》是梵高的杰出作品，是圣雷米时期最伟大的作品之一，更是我喜欢的一幅作品。鸢尾花是平凡的植物，但却被画家赋予了粲然，如同你我每一段平凡的人生可以被心灵和汗水赋予不平凡。同时令人惋惜的是，梵高也是死于坎坷情路上爱情一次次破灭的悲痛，最终用一把左轮手枪结束了自己宝贵的生命。这与鸢尾想着手解救的事情如出一辙，希望世界不再有下一个因性与爱的离断之苦而轻生的“梵高”。
#### 鸢尾属于成人社交范畴，主要服务范畴是服务于未婚群体从陌生到逐步建立亲密关系的交往和已婚群体的婚内交往。鸢尾要做什么？鸢尾致力服务于亲密关系生活。简单讲，就是帮助解决人们不知道怎么处理亲密关系的问题。展开讲，就是优化、重构、拓展婚恋生活形态，在健康的两性社交生态中通过科技手段人性化地服务用户建立、发展、加深和维系实质性的亲密关系。这横亘在其中的关键阻碍是性需求矛盾，那性需求矛盾的根因是什么？在这借用周国平先生的一段话----"婚姻的困难在于，如何在同一个异性身上把三者统一起来（性、爱情、婚姻）"。鸢尾让人在安全自由、伦理正确、循循善诱的系统生态环境中自发摒弃落后陈旧的性认知偏见，并弥合遵循理想原则的爱和遵循现实原则的婚姻这两者与遵循健康原则的性之间的现实鸿沟。在线上线下融合的交友生态中，真实的人、环境可以与线上信息实时交互；而一站式的生活解决方案是以围绕喜欢的人为核心展开的生活规划和未来期许，具体的生活方式处理交给鸢尾，主要把关注重点放在找到对的人。
#### 鸢尾的价值观驱使着我们誓要将社会重新回归人本位的思想，通过合理手段消除妨碍人们基础生活的畸高掣肘条件。房子今后必然将不再是主要问题，只要满足工作要求，就可以保障充分的生活条件。其实，房子本身给人的最大价值是稳定的未来期许，房主人将来可以稳定长久地住在一个拥有完善周边配套设施的宜居生活环境，而不是房子本身有此价值。如果得知除了孤零零的房产会被保留，周边的学校、医院、超市、交通、公园都要被拆毁，那么就算原来很值钱的房子也不会有人愿意购买。实际上，酒店旅宿也是提供的未来期许价值，酒店并不属于客户，但是客户得到了必然兑现的旅宿若干天的未来期许，就也会愿意支付货币。落实成具体案例，比如说A地区的小明与B地区的小红互许衷心，那么他们给出我们要去哪、我们要怎么过，鸢尾就会给出可行的生活解决方案，罗列出共同生活的必要条件和可选条件的清单。只要满足了清单要求，就可以保障这对爱人顺利的生活在一起。生活解决方案可以理解为一种概念自由定义而具象化的家，只要相守的人在家就在，只是住所在变化，并不一定是在某处固定房产定居。鸢尾生态的居家愿景是内任一处入户随意处落户，比如两人可以是近2年在C市住，之后3年在D市住，再往后2年E市住，40岁携手进入某国养老院等等，只要这两人没闹翻家就没散。用户也可以选择定居，或者不需要鸢尾安排，这都是用户的自由。当然了，远行会有相较本地区更多的清单要求，比如地区常识、语言、物质基础、法律知识、政治倾向与背景等。具体问题可以用量去权衡，而关乎人的主观意志的决定权则交还给用户。用户只是通过平台了解自身，在循序渐进地确立了爱情观后，借助信息工具逐步抵达想到的彼岸。

理解了自由可以从更深的层次明白忠诚的意义。自由与忠诚从不矛盾，发自内心的忠诚本身就是一种自由，而受人尊敬的自由是履行责任后的忠于真我。两者都不应走向极端，糜烂和愚忠都不美。看似“自由”的背弃和僭越不是自由，不谈忠诚至少尽责才是本分；看似“忠诚”的攫取和绑架不是忠诚，不谈自由至少退让才有底线。互相给予的自由是达成共识的一致愿望，是悲欢离合中不可避免的客观规律；而共同期许的忠诚是完全自由后的一致选择，是人情冷暖中难能可贵的一衣带水。站在笔者我的角度来看，自由和忠诚说到底都是出于爱，自由是为了不让所信之爱蒙尘，忠诚是为了不让所忠之爱锈蚀。在不产生负面影响的前提下，足够包容的文明里的自由与忠诚说到底是个美学问题，而审美终究是个轮回问题。具体什么样子才美，还要看时代与命运的角料是什么样。
