# Iris （鸢尾）

#### 这个两性社交的系统理念该叫什么，想了很久，但思来想去用鸢尾来命名挺好。鸢尾花最常用来象征爱情与友谊，也代表了好消息和爱的使者，蕴含着思念、优雅的心、使命，文中的意思也在不同颜色的鸢尾花语中有所体现。感谢能在自然的杰作里找到花语和表达意思如此接近的花，这是件幸运的事。此外，《鸢尾花》是梵高的杰出作品，是圣雷米时期最伟大的作品之一，更是我喜欢的一幅作品。鸢尾花是平凡的植物，但却被画家赋予了粲然，如同你我每一段平凡的人生可以被心灵和汗水赋予不平凡。令人惋惜的是，梵高也死于坎坷情路上爱情一次次破灭的悲痛，最终用一把左轮手枪结束了生命。这与鸢尾想着手解救的事情如出一辙，愿世界不再有下一个因性与爱的离断之苦而轻生的“梵高”。
#### 鸢尾属于成人社交范畴，主要服务范畴是服务于未婚群体从陌生到逐步建立亲密关系的交往和已婚群体的婚内交往。鸢尾要做什么？鸢尾致力于进步两性生活形态。简单讲，就是帮助解决人们不知道怎么处理亲密关系的问题。展开讲，就是优化、重构、拓展婚恋生活形态，在健康的性生态中通过科技手段人性化地服务用户建立、发展、加深和维系实质性的亲密关系。这横亘在其中的关键阻碍是性需求矛盾，那性需求矛盾的根因是什么？在这借用周国平先生的一段话----"婚姻的困难在于，如何在同一个异性身上把三者统一起来（性、爱情、婚姻）"。鸢尾让人在安全自由、伦理正确、循循善诱的性生态中自发摒弃落后陈旧的性认知偏见，从而弥合健康原则的性、理想原则的爱与现实原则的婚姻之间的现实鸿沟。在线上线下融合的交友生态中，真实的人、环境可以与线上信息实时交互；而一站式的生活解决方案是以围绕喜爱的人为核心展开生活规划和未来期许，具体的生活方式处理交给鸢尾，主要把关注重点放在找到对的人。
#### 鸢尾的价值观驱使着我们誓要将社会重新回归人本位的思想，通过合理手段消除妨碍人们基础生活的畸高掣肘条件。房屋今后必然将不再是主要问题，只要满足工作要求，就可以保障充分的生活条件。其实，住房本身给人的最大价值是稳定的未来期许，房主人将来可以稳定长久地住在一个拥有完善周边配套设施的宜居生活环境。实际上，酒店旅宿提供的也是未来期许价值，酒店并不属于客户，但是客户得到了必然兑现旅宿时限的未来期许，也同样愿意支付货币。落实成具体案例，比如说A地区的小明与B地区的小红互许衷心，那么他们给出我们要去哪、我们要怎么过，鸢尾就会给出可行的生活解决方案，罗列出共同生活的必要条件和可选条件的清单。只要满足了清单要求，就可以保障这对爱人顺利的生活在一起。生活解决方案可以理解为一种概念自由定义而具象化的家，只要相守的人在家就在，只是住所变化，并不一定是在某处固定房产的定居。鸢尾生态的居家愿景是任一处入户随意处落户，比如两人可以是近2年在C市住，之后3年在D市住，再往后2年E市住，50岁携手进入某国养老院等等，只要这两人没闹翻家就没散。当然，用户也可以选择定居，或者不需要鸢尾安排，这都是用户的自由。不过远行会有相较本地更多要求的列表清单，比如地区常识、语言、物质基础、法律知识、政治倾向与背景等。具体问题可以用量去权衡，而关乎人主观意志的决定权则交还给用户。用户只是通过平台了解自身，在循序渐进地确立了爱情观后，借助信息工具逐步抵达想到的彼岸。

## 观点
性具有了爱与自由则走向延续，失去则走向罪恶。
<br><br>
提及性，得先正视，然后承担责任。两性之中的性和自由都不是难题，难在后果的全责承担和妥善处理。性是关系的阶段结果，不是关系的应当起点；忽视从起点到结果的付出过程就会有问题，该先考虑性的责任和影响再决定是否让性发生。没有性事故的原因是性本身，而是妄图以性之名歪曲事物之实；无论这实质是罪恶或奉献都将产生矛盾，甚至与性无关的人这么做也会引发冲突。关系不管有没有性关系都只是人际关系的一种，关系之下有着怎样的情况才是问题关键，意愿和关系解读的严重出入会导致与预期大相径庭的结果，应思义悟果。理解以后，再谈性自由。
<br><br>
性自由的前提是正确认识自由，其后再诉诸于性。自由的内里是生命向外盛放的坚贞颂歌，宛如在石缝中扎根不屈生长的劲松；自由的外在是灵魂由内诉说的独特诗篇，好似每片湖泊不同模样下蕴藏的万千地貌。自由必然内含责任与规则，责任的遗弃和规则的破坏只可能毁灭自由。用欲望打破枷锁不过换来另一副枷锁，自由诞生于理想内心与有限现实的满心欢喜的握手言和之中。自由起码自愿，须要没有施加、没有潜在的威逼利诱、歪曲误导等限制因素，名不副实的自由只可能是禁锢。自由不是反目成仇，自由不是忘恩负义，相反的是真正的自由可以成就彼此的人生。自由应是成人之美的大度，自由应是海阔天空的情怀，和煦的自由之风抚掠过天空才会有乌云散尽后的澄澈。
<br><br>
理解了自由可以从更深的层次明白忠诚的意义。自由与忠诚从不矛盾，发自内心的忠诚本身就是一种自由，而受人尊敬的自由是履行责任后的忠于信念。两者都不应走向极端，糜烂和愚忠都不美。看似“自由”的背弃和僭越不是自由，不谈忠诚至少尽责才是本分；看似“忠诚”的攫取和绑架不是忠诚，不谈自由至少退让才有底线。互相给予的自由是达成共识的一致愿望，是悲欢离合中不可避免的客观规律；而共同期许的忠诚是完全自由后的一致选择，是人情冷暖中难能可贵的一衣带水。站在笔者的角度来看，自由和忠诚说到底都是出于爱，自由是为了不让所信之爱蒙尘，忠诚是为了不让所执之爱锈蚀。人生作品的挥墨，一段关系就如一次落笔，过于潦草无章无法成为佳作。在不产生负面影响的前提下，足够包容的文明里的自由与忠诚说到底是个美学问题，而审美终究是个轮回问题。具体什么样子才美，除了人生作者本身，要看时代与命运的角料是什么样。
<br><br>
简单讲，性自由是想要性就选择，不想要性就拒绝，决定权掌握在人的手中。选择了性自由不必顾虑弯弯绕绕，时代的胸怀有多宽广，人的自由就有多安全。性自由从不会胁迫任何一个性保守的人必须风流，也从不会胁迫任何一个性开放的人必须贞洁；强加于人的性观念不是性自由，而是妨碍他人公民权利的性观念绑架。性自由也是另一个角度的爱自由，可以自由选择性才可以自由拥抱爱，也才可以不受掣肘地对不爱说不。到底是选择先性更好，还是先爱更好，和潮流的规律如出一辙，时间拉长远了看是一个个大大小小的轮回。因为人多数大同，相关因素大体不变，趋势没有变化就会走向极端。而极端之处无法容人，无论是爱还是性。极端的爱让人窒息，极端的性让人腐朽。这个时代的下一个几十年到底是怎样的潮流，没有人清楚，但肯定的是走向极端的纯爱苛求会让常人无法生活。该做的应是保障大多数人的普遍基本诉求，无论是爱还是性。
<br><br>
性自由是双方自愿的，是坦诚的，不能把偏执的欲望曲解成了性自由。哪些是性自由？由性而爱、由爱而性都属于性自由，能够完全自主决定何时何地去拥抱性或放开性才算得上，能够听凭心意随时随地去选择性或拒绝性才算得上。男女彼此间都享有了这种权力，那么就可以说是性自由。哪些不是性自由？不是说谈性就得先发生性关系、或必须有性关系，也不是说谈情就得最后才发生性关系、或必须没有性关系。任何违背当事人任意一方意愿的情况都属于侵犯了人的性自由，欺骗、背叛、淫糜、野蛮等缺乏尊重的性更显得荒谬，这些明显都不是性自由。
<br><br>
做到性自由先需要了解性，还需要能把握自己的身体。性对人的伤害常常来于忽略性之外，人对性的憎恨往往源于性之内连其外，只是普遍到了涉及万象的性与它们有联系。性紧挨着生命，太平无事时并无大碍，但特殊时期的性从嬉戏到生命烙印只差一步之遥。因为性的过度或缺乏都将自我造成伤害，执迷其中或不敢面对都不过是做了思想填鸭的奴隶。只知机械复述一些经不起推敲的话语，却对由性问题不断引发的悲剧视而不见，客观的讲这是性偏见导致的思维障碍。两性敌对的起因也是由此挑起，彼此的偏见因为无法理解愈演愈烈，隔阂又因现实中往往存在的巨大认知差异加剧。这些性偏见只会带给多数常人以伤害，不善言辞的底层弱势群体做着同样的事情还要被拘捕进看守所劳改（确实作恶则罪有应得）。
<br><br>
性只是性，性不该遭受颠倒黑白的歪曲。人对性应有的态度是敬畏，而不是轻视；就如水火，不可因几日不沾水火就大肆否定，不可因戏水玩火无恙而掉以轻心。对性学的无端诋毁最终一定会成为或明或暗的不伦规则的宣泄口，婚姻制度名存实亡的惨案是肆意污蔑性学的民众不可开脱的责任。道理很直白，婚配闰余的诉求不能以正当的途径解决，就只能以冠冕堂皇的方式来解决；民众越是自私、虚伪、与权贵沆瀣一气，制度的崩毁就越是彻底，就像对饥民巧言令色地克扣粮食不会因为荒诞说辞而奏效，反而只会引发更加剧烈的次生动乱。任何对性的歪曲都只是对性学现状的不满，绝大多数抨击者的性否定是对他人远超自身，可见其并不憎恨性的存在。过去人们从事性学研究的门槛条件苛刻，但现在谁都可以为性学进步作出贡献；若有良策不应自顾藏匿，大可投身于性学变革。错的是做错性问题的人，性本身没有错。薄情寡义不是性，那是欲望的奴隶；色情更不是性，那是畸形的瘾病。性是连接感情纽带的一种，合理的性从来只会加深、稳固感情，是失控的欲望践踏了感情。问题根源不在于形式，外界制定的两性标准并不会改变善恶本质，无论禁止性或攫取性都依然可能导致罪孽深重。如今的社会现状仍存在对性的曲解、歧视，甚至有对性谬误地粗暴类比。首先，人不会因为体液的排出而获得健康，是因为交换得到了合适且适量的微生物才会健康，过量的分泌和亏出会损害健康。其次，描述对性的渴求通常用共赴巫山形容，从未听过共赴溷轩，若如此不堪的一刻春宵还能值多少？假如性是该种荒谬的比喻，那么人类根本不需要性伴侣，新陈代谢的生理需求本就独自解决。自慰的时候多是幻想爱侣给予了爱的慰藉才会有生理反应，万中无一的情况自行处理即可。两性之间的互相不尊重是催生矛盾的重大隐患。想想看，在与自己的爱人蜜意缠绵时却将与之比较，这无疑是一件侮辱人格的事情。只是这种说辞符合了受到不尊重而捍卫尊严的反击需要，性在此时正好是一把趁手又能刺伤对方心灵的利刃。但性是无辜的，有罪的是对异性轻视侮辱、施加软硬暴力的人，它不过是被当作成无力辩解又不便纷说的羔羊顶罪了。性爱的本质目的是交换着获取爱的慰藉，而不是疏泄，只有繁衍的性是交配。非要打个比方的话，性是男女情爱的食粮之一，就像农作物是生命的食粮之一。这体现在了情感、生理、特征上。情感上，和谐的夫妻生活会使情感更加牢靠，长期分居的夫妻相对容易淡了情；生理上，长期无性会导致身体出现病症、相关器官异常甚至衰败，纵欲也如暴饮暴食般会有不良后果；特征上，试想流着哈喇子地吃一颗苹果、一块鲜肉、一碗面条的情形，性伴侣也很类似。所以在历史进步的浪潮中摒弃有关性的认知糟粕，尤其是看见同样的性问题玩了套文字游戏就变得高贵时，就更加坚定了这个想法。
<br><br>
社会两性问题的普遍异样征兆是灾难的一种发端，因为这意味着种群内部团结的断裂，至少已分裂出半数的隔阂。艾滋病在人类中传播的起因就是人与黑猩猩之间的亲密交往，由Gaetan Dugas第一个把艾滋带出了非洲，再由男男的同性交往使艾滋病迅速蔓延。从1981年世界第一例艾滋病例被发现，再到艾滋病在全球肆虐流行令人闻风丧胆，一切仅仅用了20年。试问如果同类之间彼此不那么刻薄冷漠，人又怎么会和其他物种交媾？艾滋也许根本不会出现在人类之中。可距离1981年才只过了39年，人们就已经忘却了，同样的征兆又再次在世界出现。不知道这一次会诞生什么样的绝症，是和艾滋一样恐怖，还是更具杀伤力，这也许只有天知道。
<br><br>
